# Lint & Formatting

Linting und Formatting werden im Root-Verzeichnis dieses Repositories zentral verwaltet.

Bitte lege in den Unterprojekten keine lokalen Konfigdateien wie `.eslintrc*`, `eslint.config.js` oder `.prettierrc*` an, da dies zu widersprüchlichen Regeln und Merge-Konflikten führen kann.

Dev-Hinweis

- Wechsle in ein Unterprojekt: `Set-Location C:\sourcecode\uebungen_mike\<subproject>`
- Installiere Abhängigkeiten: `npm install`
- Starte den Dev-Server: `npm run dev`

Bei Ausnahmen oder speziellen Anforderungen bitte im Team absprechen.
